# Massage Therapy Smart Study Pro

## Features
- Custom Quiz Builder
- Analytics with Charts
- Spaced Repetition Visualization
- Badge System

## Setup
1. Install dependencies: `npm install`
2. Start the development server: `npm run dev`
3. Build for production: `npm run build`
